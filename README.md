# 🎨 Creative AI Studio

## AI-Powered Design Assistant

Creative AI Studio is a Generative AI tool designed to help graphic designers develop practical and creative design directions.

The tool generates design recommendations based on the user's project type, brand, business, target audience, and preferred design style.
📄 [View Project PDF](CREATIVE%20AI%20STUDIO-Wem.pdf)
## ✨ Features

- 🎯 Generates a clear design direction
- 📝 Creates headline options and supporting text
- 📌 Reminds the designer to add relevant project information
- 📐 Suggests practical layout recommendations
- 🎨 Generates a harmonious four-color palette with HEX codes
- 🔤 Recommends suitable typography
- 📊 Compares AI creativity levels using different temperature settings
- ⭐ Uses AI self-evaluation to score generated design proposals

## 🛠️ Tech Stack

- Python
- Google Colab
- Qwen 2.5
- Hugging Face Transformers
- PyTorch
- Gradio

## 🧠 How It Works

The user enters basic project information, including the project type, brand name, business type, target audience, design style, and preferred language.

The AI then generates a structured design direction with practical content, layout recommendations, color palette, typography suggestions.
